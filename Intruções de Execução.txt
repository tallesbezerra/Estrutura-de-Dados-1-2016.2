Intruções - Trabalho Prático 1 - Artefato 2
* Projeto desenvolvido na IDE Code Blocks 16.01
* Os Arquivos main.c, dicionario.c e dicionario.h devem ser
adicionados em um projeto C.
* O programa converte toda a palavra para minúscula

Intruções para o arquivo lido(palavras.txt):
	* Deve ser mantido no mesmo local onde projeto está salvo.
	* Não deve possuir mais de 1 espaço entre uma palavra e outra
	ou no final da linha.
	* Apenas serão contadas as linhas com palavras, não devendo
	haver linhas vazias no meio do texto.
	* A última linha deve ser vazia, contendo um Enter após a última
	linha com palavras.
